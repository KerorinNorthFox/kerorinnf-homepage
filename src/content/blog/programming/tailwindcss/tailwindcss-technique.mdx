## 画面のど真ん中に要素を配置する

```html
<div class="h-screen w-screen flex justify-center items-center">
  <p>center</p>
</div>
```

<Window>
  <div class="w-full h-full flex justify-center items-center">
    <p>center</p>
  </div>
</Window>

|ユーティリティクラス名|説明|
|:-|:-|
|`h-screen`|画面の高さ100%|
|`w-screen`|画面の幅100%|
|`flex`|要素を横に配置する|
|`justify-center`|要素を縦軸で真ん中に配置|
|`items-center`|要素を縦軸で真ん中に配置|

## 要素を均等に配置する

```html
<div class="flex justify-between">
  <p>A</p>
  <p>B</p>
  <p>C</p>
  <p>D</p>
</div>
```

<Window>
  <div class="flex justify-between">
    <p>A</p>
    <p>B</p>
    <p>C</p>
    <p>D</p>
  </div>
</Window>

|ユーティリティクラス名|説明|
|:-|:-|
|`justify-between`|要素を均等に配置する|

## はみ出たテキストの末尾を...で置き換える

```html
<p class="truncate">
  looooooooooooooooooooooooooooooooooooooooooooooooooooooooooong sentence
</p>
```

<Window>
  <p class="truncate">looooooooooooooooooooooooooooooooooooooooooooooooooooooooooong sentence</p>
</Window>

|ユーティリティクラス|説明|
|:-|:-|
|`truncate`|はみ出たテキストを...で置き換える|

## svgの色を変える

svgタグのfill属性とstroke属性に"currentColor"を指定することで、Tailwind CSSの`text-color`で色を変えることができる。

```html
<svg fill="currentColor" stroke="currentColor">...</svg>
```
